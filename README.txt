# When Pig Flies 2: The Rise of Batcula

## A Phaser game by Casey Albert-Hall, Alexa Drenick, Kalea Louie and Erica Solum.

### To play the game on the server, simply go to http://kalealouie.com/WhenPigFlies2/

To install from repository and run locally:

1. Make sure you have a local hosting service such as XAMPP installed and running a local web server. 
2. Clone the repository to the default folder for your localhost, typically htdocs. 
3. Simply run the folder name from your localhost.
